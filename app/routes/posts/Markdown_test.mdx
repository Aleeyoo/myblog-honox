## 1. 标题
``` Markdown
# Heading level 1

Heading level 1
===============

## Heading level 2

Heading level 2
---------------

### Heading level 3

#### Heading level 4

##### Heading level 5

###### Heading level 6
```

## 2. 字体
### 2.1. 普通：
```
I really like using Markdown.

I think I'll use it to format all of my documents from now on.
```

### 2.2. 粗体：
如：I just love __bold text__.
```
I just love **bold text**.

I just love __bold text__.

Love**is**bold
```

### 2.3. 斜体：
如：Italicized text is the *cat's meow*.
```
Italicized text is the *cat's meow*.

Italicized text is the _cat's meow_.

A*cat*meow
```
> `***text***` 或 `___text___` 为粗斜体

## 3. 引用
如：
> Dorothy followed her through many of the beautiful rooms in her castle.
```
> Dorothy followed her through many of the beautiful rooms in her castle.
>
> The Witch bade her clean the pots and kettles and sweep the floor and keep the fire fed with wood.
```

### 3.1. 嵌套：
```
> Dorothy followed her through many of the beautiful rooms in her castle.
>
>> The Witch bade her clean the pots and kettles and sweep the floor and keep the fire fed with wood.
```

## 4. 列表
如：
1. First item
2. Second item
3. Third item
    - Indented item
    * Indented item
    + Indented item
4. Fourth item
```
1. First item
2. Second item
3. Third item
    - Indented item
    * Indented item
    + Indented item
4. Fourth item
```

## 5. 代码
如：
At the command prompt, type `nano`.

``Use `code` in your Markdown file.``
```
At the command prompt, type `nano`.

``Use `code` in your Markdown file.``
```
